# Proposition de fondation du projet

## Objectif

Proposer une base propre pour démarrer le projet forum en équipe.

## Structure proposée

- `cmd/` : point d'entrée du serveur Go
- `internal/` : logique interne de l'application
- `web/` : fichiers liés à l'interface web
- `docs/` : documentation du projet

## Points déjà préparés

- README de base
- Cahier des charges
- Structure initiale du projet
- Module Go initialisé
- Convention de commits

## Décisions à prendre en équipe

- Choix des catégories du forum
- Schéma de base de données
- Répartition des fonctionnalités
- Organisation Trello
- Design général du site
- Règles de branches et Pull Requests
